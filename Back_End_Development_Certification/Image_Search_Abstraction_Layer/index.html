This is an Image Search Abstraction Layer built during a web development course on freeCodeCamp.org

The API has two query urls:

  1. jm-freecodecamp-projects.herokuapp.com/image-search-abstraction-layer/imagesearch/<yourSearchTerm>?offset=<pageNumber> (e.g. jm-freecodecamp-projects.herokuapp.com/image-search-abstraction-layer/imagesearch/cats?offset=1. The default offset is 1)
  2. jm-freecodecamp-projects.herokuapp.com/image-search-abstraction-layer/latest/imagesearch
